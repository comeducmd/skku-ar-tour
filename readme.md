## Code Convention
| 메시지 | 설명 |
|:---:|:---:|
| FEAT | 새로운 기능 추가 |
| FIX | 버그 수정|
| DOCS | 문서 추가 및 변경 |
| STYLE | 코드 포맷팅 |
| REFACTOR | 코드 리팩토링 |
| CHORE | 빌드 및 패키지 수정 |
